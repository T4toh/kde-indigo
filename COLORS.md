# Paleta de Colores - KDE Indigo

## Colores Principales

### Acentos

- **Indigo Principal**: `#673AB7` → RGB(103, 58, 183)
  - Usado para: elementos enfocados, botones primarios, enlaces
- **Purple Hover**: `#9C27B0` → RGB(156, 39, 176)
  - Usado para: estados hover, elementos interactivos

### Fondos

- **Fondo Principal**: `#181825` → RGB(24, 24, 37)
  - Ventanas principales, áreas de contenido
- **Fondo Alternativo**: `#1E1E2E` → RGB(30, 30, 46)
  - Barras laterales, paneles, botones

- **Fondo Inactivo**: `#313244` → RGB(49, 50, 68)
  - Elementos deshabilitados o secundarios

### Textos

- **Texto Principal**: `#CDD6F4` → RGB(205, 214, 244)
  - Texto normal, contenido principal
- **Texto Inactivo**: `#A1A9B1` → RGB(161, 169, 177)
  - Texto secundario, placeholders
- **Texto sobre Selección**: `#FCFCFC` → RGB(252, 252, 252)
  - Texto en elementos seleccionados

### Estados

- **Positivo**: `#A6DA95` → RGB(166, 218, 149)
  - Éxito, confirmación
- **Negativo**: `#ED333B` → RGB(237, 51, 59)
  - Error, advertencia crítica
- **Neutro**: `#FF6A00` → RGB(255, 106, 0)
  - Información, advertencia general

## Cómo Personalizar

Edita el archivo `color-schemes/KDEIndigo.colors`:

1. Busca la sección que quieres modificar (ej: `[Colors:Window]`)
2. Cambia los valores RGB
3. Guarda y reinstala el tema
4. Aplica el tema actualizado en Configuración del Sistema

## Herramientas Útiles

- **Selector de Colores KDE**: KColorChooser
- **Vista previa**: Plasma Theme Explorer
- **Generador de paletas**: <https://coolors.co>
- **Material Design Colors**: <https://materialui.co/colors>

## Tips de Diseño

1. **Contraste**: Asegúrate de que el texto sea legible (ratio 4.5:1 mínimo)
2. **Consistencia**: Usa la misma tonalidad de índigo/violeta en todo el tema
3. **Jerarquía**: Los elementos importantes deben tener más contraste
4. **Accesibilidad**: Prueba el tema con diferentes condiciones de luz
