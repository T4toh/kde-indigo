# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2026-02-02

### Añadido
- Versión inicial del tema CachyOS Indigo
- Esquema de colores completo índigo/violeta
- Tema de escritorio Plasma
- Scripts de instalación y desinstalación automáticos
- Documentación completa (README, COLORS, CUSTOMIZATION)
- Soporte para KDE Plasma 5.27+

### Cambios desde CachyOS Emerald
- Color principal: Esmeralda → Índigo (#673AB7)
- Color hover: Verde → Púrpura (#9C27B0)
- Ajustes en saturación y contraste
- Paleta de colores completamente rediseñada

### Archivos Incluidos
- `color-schemes/CachyOSIndigo.colors` - Esquema de colores
- `plasma/desktoptheme/CachyOS-Indigo-round/` - Tema Plasma
- `install.sh` - Script de instalación
- `uninstall.sh` - Script de desinstalación
- `README.md` - Documentación principal
- `COLORS.md` - Referencia de colores
- `CUSTOMIZATION.md` - Guía de personalización
- `LICENSE` - Licencia GPLv3
- `CHANGELOG.md` - Este archivo

### v1.0.0 - "Inicial"
Primera versión pública del tema. Incluye todas las funcionalidades básicas
necesarias para un tema completo de KDE Plasma. Basado en el excelente trabajo
del equipo de CachyOS con su tema Emerald.
